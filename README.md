# ODI-Result-Analysis
This repository contains the ODI Result Power-BI project.pbix, a complete interactive dashboard built to analyze international ODI cricket match results. The report highlights historical performance across teams, top-performing nations, winning patterns, toss impact, venue influence, and seasonal match outcomes.
